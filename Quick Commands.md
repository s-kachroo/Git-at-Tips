## When make a empty remote repository

    git init
    
    git add .
    
    git commit -m "Descriptive Message"
    
    git remote add origin Empty_repo_URL
    
    git push origin Branch_Name

## When cloning a remote repository

    git clone Repo_URL
    
    After making and modifying the files.
    
    git add .
    
    git commit -m "Descriptive Message"
    
    git push origin Branch_Name
